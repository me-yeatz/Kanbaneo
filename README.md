## Features

### Core Functionality
- **Kanban Board**: Organize tasks across customizable columns (To Do, In Progress, Review, On Hold, Done)
- **Drag & Drop**: Intuitive task movement between columns
- **Task Management**: Create, edit, and delete tasks with priority levels
- **Custom Columns**: Add your own columns with custom colors
- **AI Suggestions**: Get intelligent task suggestions for development and general productivity

### Design
- **Glassmorphism UI**: Modern, translucent glass-effect design
- **Purple Neon Theme**: Stunning purple gradient with glow effects
- **Dark/Light Mode**: Toggle between themes for comfortable viewing
- **Fully Responsive**: Works beautifully on desktop, tablet, and mobile

### Productivity
- **Keyboard Shortcuts**: Quick actions for power users
- **Import/Export**: Backup and restore your board as JSON
- **Local Storage**: Your data persists automatically
- **Quick Notes**: Jot down quick thoughts without creating full tasks
- **Toast Notifications**: Instant feedback for all actions
